# Practice Problem Folders

Use this area for platform-based problem solving outside the Coursera weekly folders.

## Recommended layout

- `cses/` - CSES problem set solutions
- `leetcode/` - LeetCode practice problems
- `exercism/` - Exercism exercises

## Suggested file naming

- `problem_name.py` for final solutions
- `problem_name.ipynb` for exploration or notes
- Add a short docstring with the approach and complexity when possible

## Notes

Keep the Coursera coursework in the existing `week1/`, `week2/`, `week3/`, and `week4/` folders.
