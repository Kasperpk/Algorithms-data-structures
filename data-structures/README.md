# Data Structures

Implementations and notes on fundamental data structures.

## Topics

- **Arrays & Strings** — contiguous storage, manipulation
- **Linked Lists** — singly, doubly, circular
- **Stacks & Queues** — LIFO/FIFO abstractions
- **Trees** — binary trees, BSTs, AVL, red-black
- **Heaps** — min-heap, max-heap, priority queues
- **Hash Tables** — hashing, collision resolution
- **Graphs** — representations, traversals
- **Advanced** — tries, segment trees, union-find
